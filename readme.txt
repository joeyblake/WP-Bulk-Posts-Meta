This is a plugin for doing bulk updates to wordpress posts metadata.

== Description ==

Often, when changing from one wordpress theme to another, there is post metadata that needs to be added to your existing posts in order to get them to display properly. This plugin will allow you to add post metadata to all of your posts at once, without having to open and resave every post.

== Installation ==

1. Upload `WP-Bulk-Posts-Meta` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
